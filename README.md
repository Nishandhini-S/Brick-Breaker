## Brick Breaker Console Game
          This is a simple console-based Brick Breaker game developed using Java.
          The game runs on a 7x7 grid where the player controls a ball placed on the ground and gives commands to hit bricks.
          The objective of the game is to break the bricks using directional commands.

### Features
   - 7x7 grid layout
   - Walls around the boundary
   - Ground row at the bottom
   - Ball positioned at the center of the ground
   - Bricks placed in the middle of the grid
   - Ball movement in three directions:
        - Straight (St)
        - Left Diagonal (Lt)
        - Right Diagonal (Rt)
        - Brick strength handling
        - Ball count tracking
   - Console-based user interaction

### Technologies Used
       Java
       Object-Oriented Programming concepts
       Console-based input/output
